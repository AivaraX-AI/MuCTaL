# pancan_pipe
Essentially a stream-lined version of https://github.com/HCC-data-sciences/pancancer_he_classifier, which is geared towards inference, not training.

Package that uses a neural network on densenet169 CNN trained across multiple tumor types (skin, liver, colorectal, lung) 
See 01_generate_wsi_sample_spreadsheet.ipynb for examples / instructions on use.
